Uniwhere Bot Api

Small "docs" for the Uniwhere Bot Flask App.

For all API calls, you should set the content-type in the call's header to application/json.

Authentication is supported, but in a very elementary way. When POSTing, you must to include in the call's header also an auth-token field, and set it to a certain token. To obtain the token, contact me.
