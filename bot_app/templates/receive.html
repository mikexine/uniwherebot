/receive call

The only method that is currently allowed is a POST request.

Endpoint: /receive - full url: http://uniwhere.xyz/receive.

Header Parameter Set to
content-type application/json
auth-token Your POST token
   
Body Parameter Set to
from String with the sender's ID or Username.
text Text to elaborate.

You will get a JSON object in the response to your POST request. A sample example is the following: 

{ 
  'text': 'Hello this is the answer', 
  'to': 'Send this message to this user!'
  }
